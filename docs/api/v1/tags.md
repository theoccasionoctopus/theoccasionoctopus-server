# Tags API

## List Tags



| Information | Value                                     |
| ----------- | ------------------------------------------|
| Type        | `GET`                                     |
| URL         | `/api/v1/account/ACCOUNT-ID/tags.json`    |



## Read a Tag


| Information | Value                                            |
| ----------- | -------------------------------------------------|
| Type        | `GET`                                            |
| URL         | `/api/v1/account/ACCOUNT-ID/tag/TAG-ID/.json`    |



## Edit a Tag


| Information | Value                                            |
| ----------- | -------------------------------------------------|
| Type        | `POST`                                           |
| URL         | `/api/v1/account/ACCOUNT-ID/tag/TAG-ID/.json`    |



## Create a new Tag



| Information | Value                                     |
| ----------- | ------------------------------------------|
| Type        | `POST`                                    |
| URL         | `/api/v1/account/ACCOUNT-ID/tag.json`     |
